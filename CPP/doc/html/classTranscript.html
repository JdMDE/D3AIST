Detailed Description

Transcript class.

Transcript: a class to hold all information about a transcription event from a spatial transcriptomics description file in .csv or binary format

Constructor that reads all relevant information from the .csv file and store it inside the class after doing some basic sanity checks

Parameters
idpunsigned long long: the transcription event identifier
cell_idpstring: unique identifier of the cell inside which the event transcription happens, or UNASSIGNED if happens outside any cell
overlap_nucpbool: true if the event happens inside the cell nucleus, false otherwise (even if outside cell)
feat_namepstring: the identifier of the gene or clonotype that has generated this transcription event
xpfloat: the x-coordinate of the event, in micrometers
ypfloat: the y-coordinate of the event, in micrometers
zpfloat: the z-coordinate of the event, in micrometers
qvpint: the quality value that measures the confidence on correct assignment of this event to a gene or clonotype. Values in 0..40
fov_namepstring: identifier of the region in histopathological image where the event has happened
nuc_distpfloat: distance between the event and the closest cell nucleus, or 0 if event happend inside cell nucleus
codewordpint: number asiigned to the gene or clonotype that has generated this trancription event, i.e., feat_namep.
err&bool: Test variable that returns with true if all values pass the checks and transcript has been constructed and false if not.
