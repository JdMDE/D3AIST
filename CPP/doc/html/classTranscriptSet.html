Detailed Description

TranscriptSet class.

Class TranscriptSet: a class to hold all information about a set of transcription events read from the event description file in .csv or binary format

Constructor & Destructor Documentation

◆ TranscriptSet()

TranscriptSet::TranscriptSet ( std::string tr_file_name,
bool check = false )

Constructor that takes the requested binary or .csv cell description file, reads from it all relevant information and stores it inside the class. File type (binary or .csv) is determined from its extension.

Parameters
tr_file_namestring: The name (if accessible) or complete path of the .csv or binary file
checkbool: true to check that there are no any transcritps with duplicate identifiers. This check is slow, so the parameter is by default false.
