enum class  ParamTypes {
  Boolean , String , Int , Int2 ,
  Float2 , Float4 , ListInt , ListString ,
  NoType
}
 ParamTypes documentation Enumeration with the types of values we can expect for our parameters. Int2 means 'integer integer', Float2 means 'float float', Float4 means 'float float float float'. All of these are separated by one or more blank spaces. ListInt means 'int,int,....,int', i.e., a list of comma-separated integers. Similarly for ListString. In these cases, spaces are not allowed and there must be no comma at the end.
